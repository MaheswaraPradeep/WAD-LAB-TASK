TEXT FORMATTING TAGS

There are many text formatting tags in html and some of them are listed here.

Heading tags

The h1 to h6 tags are used to define HTML headings. h1 defines the most important heading. h6 defines the least important heading.
Note: Only use one h1 per page - this should represent the small main heading/subject for the whole page. Also, do not skip heading levels - start with h1, then use h2, and so on.
